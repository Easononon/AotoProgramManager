# 三台泵恒压供水 - 西门子 S7-200 SMART（ST语言）

> 编程软件：Micro/WIN SMART V2.5+  
> 支持：LAD / STL / FBD / **ST（文本逻辑）**  
> 地址示例：V区变量 / I0.0 / Q0.0 / AIW0 / AQW0

---

## 程序说明

```
压力 ↓↓↓↓  1#泵启  2#泵启  3#泵启
压力 ↑↑↑↑  1#泵停  2#泵停  3#泵停

三台泵逐级启停，启停各30秒延时保护
```

## 全局符号定义（符号表）

| 符号 | 地址 | 说明 |
|------|------|------|
| 压力设定 | VD0 | REAL，设定压力值（MPa）|
| 实际压力 | VD4 | REAL，来自压力传感器 AIW0 |
| 滞环 | VD8 | REAL，启停差值，默认0.03MPa |
| 压力低限 | VD12 | REAL，计算值 |
| 压力高限 | VD16 | REAL，计算值 |
| 1#泵运行 | Q0.0 | DO，接触器输出 |
| 2#泵运行 | Q0.1 | DO，接触器输出 |
| 3#泵运行 | Q0.2 | DO，接触器输出 |
| 1#泵故障 | I0.0 | DI，故障输入 |
| 2#泵故障 | I0.1 | DI，故障输入 |
| 3#泵故障 | I0.2 | DI，故障输入 |

## ST程序

```st
// ============================================================
// 三台泵恒压供水 - S7-200 SMART ST
// 第一次扫描初始化
// ============================================================

// ---- 全局变量（V区）----
// VD0  = 压力设定(MPa)
// VD4  = 实际压力(MPa)
// VD8  = 滞环(MPa)
// VW10 = 备用
// VW12 = P1_ON阈值
// VW16 = P1_OFF阈值
// VD20 = P2_ON阈值
// VD24 = P2_OFF阈值
// VD28 = P3_ON阈值
// VD32 = P3_OFF阈值
// VB36 = 1#泵运行状态(BOOL)
// VB37 = 2#泵运行状态
// VB38 = 3#泵运行状态
// VB40 = 1#泵手自动
// VB41 = 2#泵手自动
// VB42 = 3#泵手自动

// ---- 子程序：泵自动控制 ----
SUBROUTINE PumpAuto
    // 入口参数（通过全局V区传递）
    // VW_IN:  模式(0=停止,1=自动)
    // VD_IN1: 实际压力(REAL)
    // VD_IN2: 设定压力(REAL)
    // VD_IN3: 滞环(REAL)

    // ---- 内部变量（临时）----
    LOCAL
        rActual : REAL;      // 实际压力
        rSet : REAL;          // 设定压力
        rHys : REAL;           // 滞环
        rP1On : REAL;          // 1#泵开启阈值
        rP1Off : REAL;         // 1#泵关闭阈值
        rP2On : REAL;
        rP2Off : REAL;
        rP3On : REAL;
        rP3Off : REAL;
        tmP1T : TON;          // 1#泵延时
        tmP2T : TON;
        tmP3T : TON;
        tmP1OffT : TON;       // 1#泵停机延时
        tmP2OffT : TON;
        tmP3OffT : TON;
    END_LOCAL

    // ---- 取输入值 ----
    rActual := VD_IN1;    // 实际压力
    rSet    := VD_IN2;    // 设定压力
    rHys    := VD_IN3;     // 滞环

    // ---- 计算阈值 ----
    // 1#泵：最近设定值，最灵敏
    rP1On  := rSet - rHys;
    rP1Off := rSet + rHys;
    // 2#泵：压力更低时启动
    rP2On  := rSet - 2.0 * rHys;
    rP2Off := rSet + 2.0 * rHys;
    // 3#泵：最后启动
    rP3On  := rSet - 3.0 * rHys;
    rP3Off := rSet + 3.0 * rHys;

    // ======================================================
    // 1#泵逻辑
    // ======================================================
    IF rActual <= rP1On AND VB40 = 1 AND NOT I0.0 THEN
        // 压力低，启动延时
        tmP1T(IN := TRUE, PT := T#30s);
        IF tmP1T.Q THEN
            VB36 := 1;           // 1#泵运行
        END_IF;
    ELSIF rActual >= rP1Off AND VB36 = 1 THEN
        // 压力高，停止延时
        tmP1OffT(IN := TRUE, PT := T#5m);
        IF tmP1OffT.Q THEN
            VB36 := 0;           // 1#泵停止
        END_IF;
    END_IF;

    // ======================================================
    // 2#泵逻辑（只有在1#泵已运行时才考虑启动2#）
    // ======================================================
    IF rActual <= rP2On AND VB41 = 1 AND VB36 = 1 AND NOT I0.1 THEN
        // 必须1#泵在运行，且压力仍然低
        tmP2T(IN := TRUE, PT := T#30s);
        IF tmP2T.Q THEN
            VB37 := 1;           // 2#泵运行
        END_IF;
    ELSIF (rActual >= rP2Off OR VB36 = 0) AND VB37 = 1 THEN
        // 压力恢复 或 1#停止 则2#停止
        tmP2OffT(IN := TRUE, PT := T#5m);
        IF tmP2OffT.Q THEN
            VB37 := 0;
        END_IF;
    END_IF;

    // ======================================================
    // 3#泵逻辑
    // ======================================================
    IF rActual <= rP3On AND VB42 = 1 AND VB37 = 1 AND NOT I0.2 THEN
        // 必须2#泵在运行
        tmP3T(IN := TRUE, PT := T#30s);
        IF tmP3T.Q THEN
            VB38 := 1;
        END_IF;
    ELSIF (rActual >= rP3Off OR VB37 = 0) AND VB38 = 1 THEN
        tmP3OffT(IN := TRUE, PT := T#5m);
        IF tmP3OffT.Q THEN
            VB38 := 0;
        END_IF;
    END_IF;

    // ---- 复位延时定时器（防止重复触发）----
    IF VB36 = 0 THEN tmP1T(IN := FALSE, PT := T#30s); tmP1OffT(IN := FALSE, PT := T#5m); END_IF;
    IF VB37 = 0 THEN tmP2T(IN := FALSE, PT := T#30s); tmP2OffT(IN := FALSE, PT := T#5m); END_IF;
    IF VB38 = 0 THEN tmP3T(IN := FALSE, PT := T#30s); tmP3OffT(IN := FALSE, PT := T#5m); END_IF;

    // ======================================================
    // 输出赋值
    // ======================================================
    Q0.0 := VB36;   // 1#泵
    Q0.1 := VB37;   // 2#泵
    Q0.2 := VB38;   // 3#泵

END_SUBROUTINE

// ============================================================
// 主程序调用
// ============================================================
MAIN
    // 读取实际压力（AIW0 = 0-27648 对应 0-1.0MPa）
    // 压力传感器：4-20mA 对应 0-1.0MPa
    VD4 := INT_TO_REAL(IN := AIW0) / 27648.0;  // 归一化

    // 如果没有AI模块，可用以下模拟值测试
    // VD4 := 0.45;  // 模拟：实际压力0.45MPa

    // 初始化设定值（第一次扫描）
    IF SM0.1 = 1 THEN   // 第一次扫描位
        VD0 := 0.5;      // 设定0.5MPa
        VD8 := 0.03;     // 滞环0.03MPa
        VB36 := 0;        // 初始停止
        VB37 := 0;
        VB38 := 0;
        VB40 := 1;        // 自动模式
        VB41 := 1;
        VB42 := 1;
    END_IF;

    // 调用泵控制子程序
    CALL PumpAuto(IN_VW := SMB0, IN_VD1 := VD4, IN_VD2 := VD0, IN_VD3 := VD8);

    // 报警：3台全开且压力持续低（30秒超时）
    // （扩展功能：可加定时器判断）

END_MAIN
```

## LAD等效逻辑（参考）

如果用LAD编写，思路相同：

```
网络1：1#泵
  I0.0(故障) --| |--| |--+--[比较 r<=P1On]--+--[TON 30s]--+(Q0.0)
                               |
  I0.0(故障) --|  |--| |--+--[比较 r>=P1Off]--+--[TON 5min]--+(Q0.0)
  
网络2：2#泵（1#运行 AND 压力低 → 2#启动）
  Q0.0(1#运行) --| |--[比较 r<=P2On]--[TON 30s]--+(Q0.1)

网络3：3#泵（2#运行 AND 压力低 → 3#启动）
  Q0.1(2#运行) --| |--[比较 r<=P3On]--[TON 30s]--+(Q0.2)
```

## IO地址表（S7-200 SMART）

| 地址 | 方向 | 说明 |
|------|------|------|
| AIW0 | AI | 压力传感器（4-20mA）|
| Q0.0 | DO | 1#泵接触器 |
| Q0.1 | DO | 2#泵接触器 |
| Q0.2 | DO | 3#泵接触器 |
| I0.0 | DI | 1#泵故障输入 |
| I0.1 | DI | 2#泵故障输入 |
| I0.2 | DI | 3#泵故障输入 |
| I0.3 | DI | 1#泵手/自动选择 |
| I0.4 | DI | 2#泵手/自动选择 |
| I0.5 | DI | 3#泵手/自动选择 |

## 调参说明

| 参数 | 地址 | 范围 | 说明 |
|------|------|------|------|
| 压力设定 | VD0 | 0.0~1.0 MPa | 触摸屏/HMI写入 |
| 滞环 | VD8 | 0.01~0.10 MPa | 越小越灵敏，越大越稳定 |
| 实际压力 | VD4 | 0.0~1.0 MPa | 只读，显示用 |

> 💡 **提示**：200 SMART 的 ST 不支持功能块实例化局部变量，定时器需放在全局V区。建议用 LAD/FBD 编写定时器逻辑，更稳定可靠。
